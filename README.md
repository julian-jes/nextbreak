# Next Break
An Android app designed for students in South Tyrol (Italy). It calculates the number of remaining school days and tracks the progress of the current school year.

## Features:
- Holiday Countdown: See how many school days remain until the next holidays.
- School Year Progress: Shows what percentage of the school year has been completed.
- Remaining Days: Displays the total number of school days remaining in the year.
- Regional Focus: Specially configured for the South Tyrolean school calendar.

## Installation:
- Download the latest apk from the [Releases](https://www.github.com/julian-jes/nextbreak/releases) page.
- Open the file on your Android device.
- If prompted, allow "Install unknown apps" to complete installation.
